# Python Study Planner

**Author:** Liam Malone

## Project Description

Python Study Planner is a command-line application that helps users manage school tasks. Users can add regular or priority tasks, view their tasks, mark tasks complete, delete tasks, and view their overall progress.

The program saves task information in a JSON file. This allows tasks to remain available after the program closes and is opened again.

This project demonstrates concepts from Chapters 8–11 of _Python Crash Course_.

## Features

- Add a regular study task
- Add a priority study task
- View all saved tasks
- Mark a task as complete
- Delete a task
- View completed and total task counts
- Save tasks automatically using JSON
- Load previously saved tasks
- Handle invalid input and file errors
- Run automated unit tests

## How to Run the Program

Open a terminal in the main project folder and enter:

```text
python main.py
```
