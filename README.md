Project Overview

This Python Study Planner is a command line Python program that keeps track of your school tasks. You can add tasks, delete them, view all your tasks, mark them complete and check your progress.

I created this project to demonstrate the Python concepts I learned in Chapters 1-7 of the book Python Crash Course.

Design Explanation

My program requires multiple pieces of information about each task, such as the subject, what the task is and whether it’s completed. So for each task, I store a dictionary, and these are all stored in a list.

Conditional statements if, elif, else are used to control what happens given the menu choice of the user. A while loop keeps the program running until we wish to quit (“q”).
